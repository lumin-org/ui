# State

The [`State`](/guide/states.md) class

## Methods

### `Listen`

Listens to changes of state within the object.

**Parameters**

- **listener:** `(changed: any) -> ()`\
  The callback to run when changes of state are made

**Returns**

- `() -> ()`

---

### `Destroy`

Destroys the state object.

**Returns**

- `void`
